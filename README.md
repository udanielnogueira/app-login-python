# App Login Python

Aplicação de login em Python usando PySimpleGUI. 

### PySimpleGUI Installation

``python -m pip install PySimpleGUI``

### PySimpleGUI Documentation

https://pysimplegui.readthedocs.io/en/latest/

- Dark Grey 13
